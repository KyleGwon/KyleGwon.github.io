About the program

Check out the code for Minesweeper.py here!

Completion Date: June 2, 2018

Program Functionality

This program uses graphics to create a remake of Minesweeper.

Planning

Before I got to making the game, I had to play it. There were notes I took on certain features. Here are some of the notes:

Functions

bombGenerator()
Uses a random number generator to create the bombs. Returns the positions of the bombs.

findBombsNear()
Checks if the tile is a bomb. Also checks the tiles around in a 1 by 1 radius. Returns whether it is a bomb, and how many bombs are around it.

createGrid()
Creates eight vertical and eight horizontal lines on the window; a 9 by 9 grid.

returnSquares()
Uses the data from bombGenerator() to make each each tile a variable set to the class Square(). Returns a list of all the tiles.

Square Class

__init__()
This class is automatically initialized when a varible is set to Square(). It sets up its instances so that there are not as many parameters later.

getRevealed()
Returns whether the tile is revealed or not.

bombsNear()
Returns 'B' if it is a bomb. If it is not a bomb, it returns the number of bombs around it.

getBomb()
Returns whether the tile is a bomb.

flag()
Flags the tile on the window.

getFlagged()
Returns whether the tile is flagger or not.
