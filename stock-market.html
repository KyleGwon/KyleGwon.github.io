About the program

Completion Date: August 15, 2018

Program Description

This was a project given to me by Franklin Templeton, an investing firm. I was told to create a program to find periods where a value stock outperformed a growth stock. The data for both of these stocks were given to me as Microsoft Excel files, and my program's results were also expected to be in an Excel file.

Program Functionality

This program parses through two sets of data containing daily stock returns from two stocks. The data is used to show the user when one stock outperformed the other. It takes in a user input of the minimum percent and the intervals in which the stocks' growth or shrinkage is measured. Using the data and user input, the program:

  • Calculates the percent difference of the growth of the two stocks over the specified interval.

  • Adds the date, whether the stock outperformed its counterpart, and the percentage of outperformance.

Planning

Before I started to write the code for this program, it was necessary to think about how I was going to get the data from the stocks. The two files containing this data stored them in columns. The critical columns were:

  • Column 1, stores the date

  • Column 2, stores the daily return

Creating

The first step, like in my other parsing programs, is to grab data from the files. The data is stored in a csv — comma separated values — format. This means that I am able to use the csv library to convert the data into a list containing each line. From here, the percent difference between returns are calculated by dividing the returns from one day by the returns from the day whose date is as far away as the user's interval specifies. After calculating the percent difference, the date associated with the percent, as well as the percent itself, are added to a list created with the purpose of housing all of this new information. I then use the reverse sorter that I created to switch the order of the list around. This data is added to a csv file, which can be opened in Microsoft Excel or Google Sheets.

Functions

reverseSort()
Reverses the order of items in a given a list.

compare()
Returns a list containing the dates, whether the stock outperformed the other, and the percent differences.

returns()
Returns the comparison between the two sets of data, each with one stock's dates and prices, utilizing the function compare().

Data Files

value.csv
Contains the data for the value stock.

growth.csv
Contains the data for the growth stock.

percents.csv
The file that the data is added to.
